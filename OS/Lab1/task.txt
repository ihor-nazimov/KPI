Лабораторная работа № 1

Выдача содержимого каталога

Написать программу, выдающую содержимое указанного (по умолчанию текущего) каталога подобно команде ls в следующем виде:
•	тип файла - один из символов {-,d,c,b,l,s,p}; /* switch */
•	права доступа - как в команде ls; /* if */
•	количество жестких ссылок;
•	имя владельца - для выяснения имени владельца по его номеру воспользуйтесь функцией getpwuid;
•	имя группы - для выяснения имени группы по ее номеру воспользуйтесь функцией getgrgid;
•	размер файла в байтах;
•	время последней модификации в строковом виде (функция ctime);
•	индекс;
•	имя файла.
Примечания. Имя каталога для выдачи его содержимого указывается в командной строке. Если имя не указано, выдается содержимое текущего каталога. Если указанный (текущий) каталог не удалось открыть, то выдать сообщение об ошибке. В случае возникновения других ошибок также необходимо выдавать соответствующие сообщения. Для получения информации о файле следует использовать функцию lstat, а не stat. Если имя пользователя (группы) установить не удалось, то на месте имени пользователя (группы) выводить соответствующий идентификатор.
